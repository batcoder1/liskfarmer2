# LISK FARMER

Make deposits, borrows, repayments and withdrawals on the Lisk network.

## Run Locally

Create a .env. file with your privateKey
```bash
  PRIVATE_KEY=0x....
```
Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
